## Commands

```bash
pnpm dev          # Start dev server with concurrent Tailwind watch + Vite (localhost:5173)
pnpm build        # Generate tw-version, format, then Vite production build
pnpm format       # Prettier format (CSS, PCSS, JS, JSX)
pnpm cc           # Check formatting with Prettier (no write)
pnpm spell        # CSpell spell check
pnpm release      # Bump patch version, build, and deploy to Vercel
```

## Architecture

Single-page React app for experimenting with color palettes and random value sets. Derived from funnel-r3t with the SVG funnel chart removed.

**State flow:**
- `src/utils/store.js` — zustand + persist store; single source of truth persisted to `localStorage` under `funnel-r3t::app-data`
- `actions` (exported from store) — call `useStore.getState()`/`setState()` directly
- Components subscribe with `useStore(s => s.field)` selectors

**Key components:**
- `Controls.jsx` — refresh/reset buttons and click counter
- `Colors.jsx` — color palette picker
- `Values.jsx` — displays the current random value set

**Key utilities:**
- `src/utils/store.js` — zustand store + actions (`refresh`, `reset`, `setPick`)
- `src/utils/funnelDefs.js` — defines 5 sections; used only for `.map()` to generate 5 random values
- `src/utils/namedColors.js` — 20 color palettes (clrs.cc + custom)
- `src/utils/tw-version.js` — auto-generated by the `.generate` script; do not edit manually

**Styling:**
- Tailwind CSS 4 with PostCSS; input is `src/styles/input.pcss`, compiled output is `src/styles/app.css`
- Custom color plugin `proto-tailwindcss-clrs` maps named palettes to Tailwind utilities
- Tailwind preflight is disabled in `tailwind.config.js`

**Deployment:** Vercel — `pnpm release` bumps patch version, builds, and deploys.
